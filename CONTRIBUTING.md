# Contributing

このリポジトリは Flutter Web 製のポートフォリオ履歴書です。外部 Pull Request、一般的な機能要望、サポート依頼、表示文言の改善提案は受け付けていません。

Issue は、依存更新や公開リポジトリ運用上の衛生報告のために限定的に有効化しています。公開 Issue、Pull Request、コメント、ログ、スクリーンショット、commit message には、秘密情報、認証情報、アクセストークン、private URL、ローカルマシンの絶対パス、個人メールアドレス、非公開の個人情報を含めないでください。

脆弱性の疑いがある内容は public Issue に投稿せず、GitHub Security Advisories から非公開で報告してください。

依存更新 Pull Request は Dependabot / Renovate などの自動更新ツールから作成され、リポジトリ owner がレビューします。GitHub Actions と workflow 依存の更新は、SHA pinning と selected actions の設定を維持したまま手動で確認します。
