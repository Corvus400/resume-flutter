# Security Policy

このリポジトリは静的配信の Flutter Web ポートフォリオです。

脆弱性の疑いがある内容は、このリポジトリの GitHub Security Advisories から報告してください。非公開で扱うべき詳細を public issue に投稿しないでください。

報告には、不要な秘密情報、アクセストークン、ローカルマシンの絶対パス、非公開の個人情報を含めないでください。

## Scope

対象:

- Flutter Web アプリケーション
- GitHub Pages deploy workflow
- `content/resume/` から生成される公開履歴書コンテンツ

対象外:

- 一般的なサポート依頼
- 表示文言の改善提案
- 外部サービスや GitHub 自体の脆弱性

## Repository Operation

このリポジトリはポートフォリオ用途です。外部 PR は受け付けません。依存更新とセキュリティ通知のため Issue / Dependabot / GitHub Security 機能を利用します。
